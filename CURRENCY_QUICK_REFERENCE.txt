💻 HOW TO USE IN CODE

1. Import the hook:
   import { useCurrency } from "@/contexts/CurrencyContext";

2. Use in component:
   const { formatPrice, currency, setCurrency } = useCurrency();

3. Format prices:
   <p>{formatPrice(10000)}</p>  // Shows: ₹10,000 or $120 or €110 etc.

4. Get current currency:
   console.log(currency);  // "INR", "USD", etc.

5. Change currency programmatically:
   setCurrency("USD");
