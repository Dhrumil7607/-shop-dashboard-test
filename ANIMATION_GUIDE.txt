🔄 CURRENCY CHANGE FLOW

User Action: Click Currency Selector → Select "USD"
         ↓
LocalStorage Updates: slb_currency = "USD"
         ↓
CurrencyContext Updates: currency = "USD"
         ↓
formatPrice() Re-calculates:
   1500 INR × 0.012 USD rate = $18
         ↓
React Re-render: Component sees new price
         ↓
Animation Triggers:
   {formatPrice(1500)} updates from "₹1,500" to "$18"
         ↓
Framer Motion Animates:
   Initial: { opacity: 0.5, scale: 0.8 }
   Animate: { opacity: 1, scale: 1 }
   Duration: 400ms
         ↓
User Sees: Smooth price transition!

⚙️ IMPLEMENTATION DETAILS

Each animated price uses:

<motion.span
    key={`unique-key`}              // Forces re-animation
    initial={startingState}          // How it starts
    animate={endingState}            // How it ends
    exit={exitState}                 // When leaving
    transition={{
        duration: milliseconds,      // 400-500ms
        ease: "easeInOut"            // Easing function
    }}
>
    {formatPrice(priceInINR)}        // Dynamic price
</motion.span>
